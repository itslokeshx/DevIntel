### âš¡ Quick Start

**1. Clone & Install**
```bash
git clone https://github.com/itslokeshx/DevIntel.git
cd DevIntel
```

**2. Backend Setup**
```bash
cd backend && npm install
npm start
```

**3. Frontend Setup**
```bash
cd ../frontend && npm install
npm run dev
```

**4. Environment Secrets**
Create a `.env` file in `backend/` with your keys:
```env
MONGODB_URI=...
GITHUB_TOKEN=...
OR_API=...
```
